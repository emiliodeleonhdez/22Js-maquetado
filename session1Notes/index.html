Semantica HTML5

Con el lanzamiento del HTML5 surgieron nuevos elementos enfocados en aumentar el poder semántico y estructural del lenguaje

header

El elemento de HTML Header header representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes.

Este texto está contenido por un div que a su vez, está contenido por una etiqueta header

nav

El elemento HTML nav representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.

Este texto está contenido por un div que a su vez, está contenido por una etiqueta nav

main

El elemento HTML main representa el contenido principal del body de un documento

Este texto está contenido por un div que a su vez, está contenido por una etiqueta main

section

El elemento de HTML section section representa una sección genérica de un documento. Sirve para determinar qué contenido corresponde a qué parte de un esquema

Este texto está contenido por un div que a su vez, está contenido por una etiqueta section

article

La etiqueta article especifica contenido independiente

Es normalmente utilizado por la naturaleza de su etiqueta para posts en blogs, historias, etc

Este texto está contenido por un div que a su vez, está contenido por una etiqueta article

aside

El elemento HTML aside representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Estas secciones son a menudo representadas como barras laterales o como inserciones

Este texto está contenido por un div que a su vez, está contenido por una etiqueta aside

footer

El Elemento HTML Footer footer representa un pie de página para el contenido de sección

Este texto está contenido por un div que a su vez, está contenido por una etiqueta aside
